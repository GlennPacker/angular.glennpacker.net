The Basic Angular Module

The angular module is where all items for the feature should be regisgered.

Declarations

This is where you register each of the components taht make up the feature.

Imports

This is where all services should be registered including routing and ngrx effects and state.

Exports

This section of the module states any components that could be used by another module consuming this resource. For instace witha custom component library you would import each of the components and then export them so anther module can just import the module to access the library. 

    
        import {{'{'}} NgModule {{'}'}} from '@angular/core'; 
        import {{'{'}} CommonModule {{'}'}} from '@angular/common'; 
        import {{'{'}} MyComponent {{'}'}} from './components/my-component.component'; 
        
        @NgModule({{'{'}} 
            declarations: [ 
                MyComponent 
            ], 
            imports: [ 
                CommonModule 
            ], 
            exports: [ 
                MyComponent 
            ]
        {{'}'}})
        export class ComponentsModule {{'{}'}}
