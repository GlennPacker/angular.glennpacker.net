Transclusion / Content Projection

Transclusion/Content Projection allows the creation of component where several areas are templates with holes. The holes the calling component can then use to insert markup.

Calling component 
            
    <app-my-component
        title="Card Title"
    >
        <h4>Slot Content</h4>
        <p>lots more markup.</p>
    <\app-my-component>
Component using transclusion 
            
    <div class="card">
    <div class="card-header text-white bg-primary mb-3">
    {{ '{' }}{{ '{' }}title{{ '}' }}{{ '}' }} 
    </div>

    <div class="card-body">
        <ng-content></ng-content>
    </div>
    </div>

lots more markup.

Multi-slot

It is possible to insert into several places using transclusion. There are several ways to do this using the 'select' attribute.

The slots here are named 'title' on the span and 'body' on the div

Calling component 
            
        <app-my-component>
            <span title>Card Title</span>
        
            <div body>
                <h4>Slot Content</h4>
                <p>lots more markup.</p>
            </div>
        <\app-my-component>
Recieving component's template 
            
        <div class="card">
            <div class="card-header text-white bg-primary mb-3">
            <ng-content select="[title]"></ng-content>
            </div>
            
            <div class="card-body">
                <ng-content select="[body]"></ng-content>
            </div>
        </div>
